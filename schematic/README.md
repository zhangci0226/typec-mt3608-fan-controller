# 嘉立创EDA项目文件

本项目使用嘉立创EDA进行原理图和PCB设计。

## 📁 文件说明

- `typec_power.sch.pdf` - 原理图PDF文件
- `typec_power.png` - 原理图预览图片
- `typec_power.json` - 嘉立创EDA项目文件（如果有）

## 🛠️ 如何打开项目

### 方法一：使用嘉立创EDA在线版
1. 访问 [嘉立创EDA官网](https://lceda.cn/)
2. 登录您的账户
3. 导入项目文件或重新创建

### 方法二：使用嘉立创EDA客户端
1. 下载并安装嘉立创EDA客户端
2. 打开项目文件
3. 进行编辑和修改

## 📋 设计要点

### 原理图设计
- 使用MT3608升压芯片
- Type-C接口设计
- MOSFET控制风扇和LED
- 拨动开关控制电源

### PCB设计
- 40x40mm紧凑设计
- 2层板设计
- 考虑散热布局
- 预留安装孔位

## 🔧 修改建议

如需修改设计：
1. 在嘉立创EDA中打开项目
2. 修改原理图或PCB
3. 重新生成Gerber文件
4. 更新BOM清单
